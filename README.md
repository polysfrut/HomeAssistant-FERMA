# Home Assistant - Брудер для перепелиной фермы

Минимальная конфигурация для управления брудером.

## Структура файлов

- `configuration.yaml` - основной файл конфигурации
- `packages/brooder.yaml` - конфигурация брудера
- `ui-lovelace.yaml` - dashboard
- `esphome/waveshare.yaml` - конфигурация ESPHome контроллера

## Установка

1. Скопируйте файлы в директорию `/config` Home Assistant
2. Перезагрузите Home Assistant
